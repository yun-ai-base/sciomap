# Sciomap — 科学演化星图 设计方案

## 核心理念

**满天星 + 结构聚合：** 远看是学科星图（星光亮度=影响力），近看是演化树（时间轴+学科分支），点进去是问题链（前人卡在哪→谁突破→吵了什么→开了什么新路）。

## 项目结构

```
D:\applications\AI_files\sciomap\
  ├── index.html
  ├── 方案.md
  └── data\
      └── scientists.json
```

纯静态，GitHub Pages 托管，跨端访问。跟 philomap 一样单 HTML + CDN D3。

## 可视化方案

### 布局原理

- **forceX(birth)** → 节点按出生年排列，形成时间轴
- **forceY(discipline)** → 按学科纵向分层，形成演化树的"冠层"
- 跨学科节点（如牛顿=物理+数学）在两层之间悬浮
- **连线**：实线=师承，虚线=影响，点线=推翻/冲突（带闪烁动画）

### 中英文切换

继承 philomap 的 i18n 模式：
- 所有字段双语言：`name`/`nameEn`、`summary`/`summaryEn`
- 右上角语言切换开关
- `localStorage` 记住偏好
- 人名、学科、时代、子学科全部中英双语

### 移动端适配

- 学科分层在手机上拍扁为单层，用颜色区分
- 底部迷你时间滑块
- 筛选面板全屏遮罩弹出
- 详情面板底部抽屉式
- 字体大小设置（小/中/大）
- 纯触摸交互，无 hover 依赖

### 大屏 vs 小屏差异

| 元素 | 大屏 | 小屏 |
|------|------|------|
| 学科分层 | 纵向 7 层 | 拍扁单层，颜色区分 |
| 树枝标注 | zoom in 显示 | 收起 |
| 瓶颈/问号节点 | 小灰点 | 放大触点 |
| 筛选面板 | 侧边固定 | 底部全屏 |
| 详情面板 | 右侧抽屉 | 底部抽屉 |
